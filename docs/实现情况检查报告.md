## 已修复的问题

### 问题1：Mapper扫描配置冲突 ⚠️ → ✅

**问题描述：**
- `MybatisPlusConfig`扫描`com.xx.xianqijava.mapper`
- `UidGeneratorConfig`扫描`com.baidu.fsg.uid.worker.dao`
- 两个`@MapperScan`可能冲突

**解决方案：**
- ✅ 将两个包合并到`MybatisPlusConfig`的`@MapperScan`中
- ✅ 从`UidGeneratorConfig`中移除`@MapperScan`注解

**修改后配置：**
```java
@Configuration
@MapperScan({"com.xx.xianqijava.mapper", "com.baidu.fsg.uid.worker.dao"})
public class MybatisPlusConfig {
    // ...
}

@Configuration
public class UidGeneratorConfig {
    // 不再包含@MapperScan
}
```
