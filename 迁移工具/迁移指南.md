## 🚀 快速开始

### 第一步：Mac 上导出

```bash
# 进入迁移工具目录
cd /Users/nickxiao/11project/XianQiJava/迁移工具

# 运行导出脚本
./export_for_migration.sh
```

**导出内容：**
- MySQL 数据库（表结构 + 数据）
- 项目源代码
- 上传的图片文件
- Windows 导入脚本

**生成的文件：**
```
xianqi_backup_YYYYMMDD_HHMMSS.tar.gz
```

---

### 第二步：传输到 Windows

将生成的 `.tar.gz` 文件复制到 Windows 机器，可以使用：

| 方式 | 说明 |
|------|------|
| U盘 | 直接复制 |
| 网络共享 | SMB/AFP 共享文件夹 |
| 云盘 | 上传后下载 |
| 微信/QQ | 文件传输 |

---

### 第三步：Windows 上导入

#### 3.1 解压文件

使用以下工具解压 `.tar.gz` 文件：
- **7-Zip**（推荐，免费）：https://www.7-zip.org/
- **WinRAR**：https://www.rarlab.com/
- **Windows 自带**：右键 → 解压（Windows 10/11）

#### 3.2 运行导入脚本

双击运行 `import_windows.bat`

脚本会自动：
1. 检查 Java 环境
2. 检查 MySQL 连接
3. 创建数据库
4. 导入数据
5. 配置项目
6. 生成启动脚本

确保以下命令可以在 CMD 中运行：


如果提示"不是内部或外部命令"，需要将对应软件的 bin 目录添加到系统 PATH。

## 🔧 配置修改

导入完成后，检查并修改配置文件：

```
files\XianQiJava\src\main\resources\application.yml
```

### 需要检查的配置项

```yaml
# 数据库配置（如果密码不同）
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/XianQi?useUnicode=true&characterEncoding=UTF-8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true&connectionCollation=utf8mb4_unicode_ci
    username: root
    password: 123456              # ← 改成你的 MySQL 密码

# Redis 配置（如果设置了密码）
  data:
    redis:
      host: localhost
      port: 6379
      password:                   # ← 如果 Redis 有密码，填写这里

# 文件上传路径（Windows 格式）
file:
  upload:
    path: C:/xianqi-uploads       # ← Windows 路径，使用 / 或 \\

# 应用基础 URL
app:
  base-url: http://localhost:8080
```

## ❓ 常见问题

### Q1: MySQL 连接失败

**现象：** `Access denied for user 'root'@'localhost'`

**解决方案：**
1. 检查 MySQL 服务是否启动
   - Win + R → `services.msc` → 找到 MySQL
2. 检查密码是否正确
3. 尝试命令行登录：`mysql -u root -p`

---

### Q2: 端口 8080 被占用

**现象：** `Port 8080 was already in use`

**解决方案：**

```cmd
# 查看占用进程
netstat -ano | findstr 8080

# 结束进程（PID 从上面命令获取）
taskkill /F /PID <PID>

# 或者修改端口
# 编辑 application.yml，修改 server.port
```

---

### Q3: Redis 连接失败

**现象：** `Unable to connect to Redis`

**解决方案：**

1. 检查 Redis 是否启动
   ```cmd
   redis-cli ping
   ```
   应返回 `PONG`

2. 如果不需要缓存，禁用 Redis：
   ```yaml
   spring:
     autoconfigure:
       exclude:
         - org.springframework.boot.autoconfigure.data.redis.RedisAutoConfiguration
   ```

---

### Q4: Java 版本不对

**现象：** `Unsupported class file major version 61`

**解决方案：**

项目需要 Java 17，请安装正确版本：

```cmd
# 检查当前版本
java -version

# 应显示类似
java version "17.0.x"
```

下载地址：https://adoptium.net/

---

### Q5: 找不到或无法加载主类

**现象：** `Could not find or load main class`

**解决方案：**

```cmd
# 清理并重新编译
mvnw.cmd clean package -DskipTests
```
